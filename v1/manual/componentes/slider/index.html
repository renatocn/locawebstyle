Utilização

$('.carousel').carousel();

Você pode definir os intervalos com o atributo data-interval ou quando o slider irá pausar:

Nome type Padrãoth> Descrição
interval number 5000 A quantidade de tempo de delay entre o ciclo automático do ítem. Se for false, o carousel não irá circular automaticamente.
pause string "hover" Pausa o ciclo do carousel no mouseenter e volta o ciclo no mouseleave.

Métodos

Para utilização em scripts personalizados, você pode utilizar os seguinte métodos para controlar o carousel/slider.

Tipo Descrição
.carousel('cycle') Faz o carousel circular da esquerda para direita.
.carousel('pause') Pára o carousel.
.carousel(number) Circula o carousel para um item em particular. Similar a uma array.
.carousel('prev') Circula o carousel para o ítem anterior.
.carousel('next') Circula o carousel para um ítem a frente.

Eventos

Eventos que são ativados quando o carousel/slider for utilizado.

Evento Descrição
slide Este evento dispara imediatamente quando a instacia do método é chamada.
slid Este evento é disparado quando a transição do carousel for completada.
