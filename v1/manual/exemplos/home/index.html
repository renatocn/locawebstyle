Dicas

  • O que são os erros de recebimentos ou bounces?

    Bounces são tipos diversos de problemas que impedem que um contato receba a mensagem enviada. Por exemplo: e-mail inválido, caixa postal cheia, lentidão no servidor do destinatário, etc

  • Evite expressões "clique aqui"

    Evite usar textos que podem ser interpretados como SPAM. Veja a lista com as expressões que você deve evitar:

    • Clique aqui
    • Mailmkt
    • Crédito
    • Grátis
    • Curso
    • De/Para sua empresa
  • Não use apenas uma imagem para enviar todo o seu conteúdo

    Evite utilizar uma única grande imagem como conteúdo do email, pois isso é considerado prática de spam. Em vez disso, prefira quebrar a imagem em pedaços menores caso seu conteúdo seja apenas esta imagem.
