Clique automático

Ativando elementos pela URL

Através da url ative elementos como tabs específicas, tooltips e outros componentes.

Para usar este comportamento, basta adicionar na url o id do elemento que ativa o comportamento desejado, no formato #!/#ID.

Após o carregamento da página, o link/elemento na página que possuir o atributo href="#ID" ou data-target="#ID" será ativado com a simulação de um click via javascript.

Esta funcionalidade pode ser utilizada para:

  • Ativar uma tab específica
  • Simular um clique em um botão ou link
  • Iniciar uma tela com um collapse aberto
  • Iniciar uma tela com um dropdown aberto
  • Abrir um modal
  • Abrir um popover
  • Iniciar um tour guiado após o carregamento da página
