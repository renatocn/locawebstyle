Calendário

Calendário e para facilitar a selação de datas

Para usar calendário no campo desejado, basta usar o seletor .datapicker no campo.

1
2
3
4
5
6
<form action="" class="ls-form ls-form-horizontal" data-ls-module="form">
  <label class="ls-label col-md-3">
    <b class="ls-label-text">Data</b>
    <input type="text" name="cel2" class="datepicker"  >
  </label>
</form>

Utilizamos o plugin Pikaday para o calendário do Locawebstyle, preé-configurado para pt-br.

Para iniciar um novo datapicker com javascript, utilizando as mesmas configurações, use, no javascript:

1
locastyle.datepicker.newDatepicker('#id ou .classe do elemento');

Todas as demais configurações e métodos do plugin Pikaday podem ser utilizados, conforme descrito na sua documentação.
