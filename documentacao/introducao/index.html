Use pelo nosso servidor

Nós disponibilizamos as versões do Locaweb Style em nosso servidor de alta disponibilidade para qualquer desenvolvedor. Basta usar os links abaixo em seu projeto. A única de depêndencia é a jQuery.

1
2
3
4
5
<!-- Coloque o CSS no seu HEAD -->
<link rel="stylesheet" type="text/css" href="//assets.locaweb.com.br/locastyle/3.0.0/stylesheets/locastyle.css">

<!-- Coloque o JS no seu FOOTER, logo depois da jQuery -->
<script src="//assets.locaweb.com.br/locastyle/3.0.0/javascripts/locastyle.js"></script>

Se você quiser usar a versão mais atual que existe, use os assets em edge. Veja abaixo um exemplo:

1
2
3
4
5
<!-- Coloque o CSS no seu HEAD -->
<link rel="stylesheet" type="text/css" href="//assets.locaweb.com.br/locastyle/edge/stylesheets/locastyle.css">

<!-- Coloque o JS no seu FOOTER, logo depois da jQuery -->
<script src="//assets.locaweb.com.br/locastyle/edge/javascripts/locastyle.js"></script>

Sempre que há uma atualização, esses assets são atualizados automaticamente.
