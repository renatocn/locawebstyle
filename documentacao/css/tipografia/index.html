Tipografia

Conheça a tipografia do Locaweb Style

Você sempre vai escrever títulos usando as tags do HTML h1, h2, h3, h4, h5 e h6. As tags sozinhas não tem estilo algum, por isso você precisa usar as classes .ls-title-1, .ls-title-2, .ls-title-3, .ls-title-4, .ls-title-5, .ls-title-6.

As classes são interessantes para combinar estilo e semântica. Você pode querer que um H5 tenha o mesmo tamanho e estilo da classe .title-1.

h1 Locastyle Heading .ls-title-1

h2 Locastyle Heading .ls-title-2

h3 Locastyle Heading .ls-title-3

h4 Locastyle Heading .ls-title-4

h5 Locastyle Heading .ls-title-5
h6 Locastyle Heading .ls-title-6
