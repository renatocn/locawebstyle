Cores do Tema

Insira cores no seu projeto.

Existem uma série de temas de cores base usados no Locaweb Style. Essas cores são adicionadas conforme vamos implementando novos produtos na Locaweb. O tema de cor é aplicado via classe no elemento html.

Classe da cor Cor
ls-theme-green Tema verde
ls-theme-light-green Tema verde claro
ls-theme-blue Tema Azul
ls-theme-gold Tema dourada
ls-theme-gray Tema cinza
ls-theme-orange Tema laranja
ls-theme-light-red Tema Vermelho claro
