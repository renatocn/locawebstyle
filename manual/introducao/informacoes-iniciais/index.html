Dependências do Locaweb Style

Temos algumas dependências e informações importantes antes da diversão.

O Locaweb Style precisa das dependências abaixo para funcionar perfeitamente. Sempre recomendamos ter a versão mais atual destes plugins.

Bootstrap

Os estilos e comportamentos foram baseados no Bootstrap, framework criado pelo Mark Ott e Jacob Thornton para agilizar a criação de interface do Twitter.

Se você não conhece, visite o github do projeto e conheça já!

Curiosidade

O Bootstrap utiliza o Normalize, que é um CSS Reset bastante completo. Ele padroniza e compatibiliza alguns problemas de textos e valores padrões em elementos do HTML. O Normalize foi criado pelo Nicolas Gallagher and Jonathan Neal.
Dê uma olhada no projeto

JQuery

O JQuery é o framework mais utilizado para controle e manipulação de comportamentos de elementos do HTML. Acho que ele dispensa apresentações. Link do projeto

Desenvolvimento

Para extender e contribuir com o Locastyle, é necessário conhecer algumas tecnologias e práticas:

  • Middleman: gerador de sites estáticos, feito em Ruby, usado para gerar e compilar os arquivos;
  • SASS: um precompilador de CSS, que utiliza uma sintaxe diferente e adiciona algumas funcionalidades, facilitando a criação e manutençãoo de projetos com CSS muito extensos;
  • COMPASS: uma biblioteca de funções de SASS que agiliza o desenvolvimento e fornece funções (@mixins) prontos que a comunidade já conhece e aprova;
  • Além do básico, HTML, CSS e Javascript
