Tabelas

Seguimos os padrões do bootstrap com um toque de usabilidade.

A tabela é elemento comum e bastante usado para listagem de conteúdo com muitas informações agregadas. Nossa tabela tem uma série de ações que extendem as funcionalidades da simples tabela que o Bootstrap dá.

Tabela básica

A tabela funciona totalmente igual ao do Bootstrap, incluse as classes. Fizemos uma classe customizada para fazer nosso estilo, essa classe é a table-style.
