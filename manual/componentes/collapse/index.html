Collapse

Agrupando grandes quantidades de informações.

Collapses são usados para organizar grandes quantidades de conteúdo que trazem uma informação ou comportamentos do sistema para a interação com o usuário. Veja os exemplos abaixo:

Exemplo de layout com formulário

Você pode colocar formulários da maneira que quiser aqui dentro.

Um formulário de cadastro:

<div class="collapse-box">
	<div class="panel-heading">
		<a data-toggle="collapse" href="#collapseOne">
			<h4 class="panel-title">Título do Collapse</h4>
			<p>Descrição do collapse, se houver.</p>
		</a>
	</div>
	<div id="collapseOne" class="panel-collapse collapse">
		<div class="panel-body">

			<p>Aqui é onde você coloca o conteúdo do Collapse</p>

		</div>
	</div>
</div>

O collapse é responsivo, logo se adapta de acordo com o tamanho da tela do usuário. Como todos elementos do Locaweb Style são responsivos, os elementos que ficarão dentro do collapse não devem que quebrar na mudança de largura de tela.

Utilização com atributos

Dentro do elemento .panel-heading deve haver qualquer elemento HTML com os seguintes atributos: data-toggle="collapse", que avisa que aquele trigger se refere a um collapse e data-target="#id" que define o collapse alvo. Atenção: Se você estiver usando um link como trigger, o atributo data-target é dispensável. Nesse caso você pode colocar o alvo direto no atributo href="#id" do link.

Utilização com Javascript

Ative um collapse manualmente usando o código abaixo:

$(".collapse").collapse();

Métodos

Ativa seu conteúdo como um elemento de collapse.

$('#myCollapsible').collapse({
  toggle: false
});
método descrição
.collapse('toggle') Alterna entre show e hide um elemento de collapse.
.collapse('show') Abre um elemento collapse.
.collapse('hide') Fecha um elemento collapse.

Eventos

Evento Descrição
show.bs.collapse Este evento é disparado imediatamente quando a instância show é chamada.
shown.bs.collapse Este evento é disparado quando um elemento de collapse está visível para o usuário. Irá esperar pela transição do CSS ser completada.
hide.bs.collapse Este evento é disparado imediatamente quando o método hide for chamado.
hidden.bs.collapse Este evento é disparado quando um elemento de collapse ficou escondido para o usuário. Ele espera até as transições do CSS terminarem. 
$('#myCollapsible').on('hidden.bs.collapse', function() {
// Faça algo
});
